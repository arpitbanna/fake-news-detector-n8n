## 🎯 Project Overview

This is a full-stack AI-powered system that detects fake news using a **multi-agent architecture**. 

The system deeply analyzes news text using:
* Sentiment analysis
* Credibility scoring
* AI-based reasoning

It then combines the results using custom logic to produce:
* **Final verdict**: Real / Fake / Unclear
* **Confidence score**
* **Explainable AI (reasons)**

## ✨ Features

* 🧠 **Multi-agent AI pipeline:** Utilizes 3 concurrent LLM chains for robust validation.
* ⚖️ **Final verdict logic:** Accurately evaluates text into 3 classes (Real, Fake, Unclear).
* 📊 **Confidence score:** Outputs a calculated percentage (0–100) on verdict reliability.
* 🎭 **Sentiment + intensity analysis:** Detects underlying tone (Positive, Negative, Neutral) and its intensity.
* ⭐ **Credibility scoring:** Rates the likely authenticity of the source (0–10).
* 🔍 **Explainable AI:** Dynamically maps a concrete reason list generated by the AI to explain the verdict.
* ⚡ **Real-time analysis:** Communicates via a fast REST API webhook.
* 🎨 **Modern UI:** Features a premium glassmorphic, responsive, and animated user interface.

---

## 🏗️ System Architecture

The analysis pipeline operates in an asynchronous, multi-agent format:

1. **Input:** The user enters a news headline or article snippet in the React frontend.
2. **Transit:** A POST request containing the text is sent securely to an n8n webhook.
3. **Execution:** n8n triggers multiple AI models concurrently:
   * **Sentiment Model:** Evaluates the emotional lean.
   * **Credibility Model:** Checks for factual likelihood and structural integrity.
   * **Verdict Model:** Generates an ultimate ruling.
4. **Processing:** Results are merged and processed through custom JavaScript node logic within n8n.
5. **Response:** A final structured JSON array is returned to the frontend.
6. **Rendering:** React dynamically displays the results, mapping metrics and explanations instantly.
